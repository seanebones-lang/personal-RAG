PersonalRAGVault is a local-first RAG system that ingests personal data from downloads (emails, chats, invoices, codebases, tweets), embeds it with a lightweight 0.6B model on CPU, stores in a vector database, and enables natural language queries via any local LLM like Ollama. Optimized for MacBook M1 with CPU inference, it supports easy hosting. Retrieve relevant data without LLM memory limits, e.g., 'find important tweets on machine learning, DALL-E, AI subs'.
