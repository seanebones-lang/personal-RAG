# PersonalRAGVault v1.2.0 Release Notes

**Release Date:** June 2, 2026

This is a significant feature release that dramatically expands the kinds of personal data you can bring into your local knowledge base, while also improving the day-to-day experience for both casual and power users.

---

## Highlights

- **Major expansion of personal data sources** — You can now ingest WhatsApp chats, Notion pages, calendar data, and more bookmark services directly.
- **Better chat experience in the UI** — Answers now stream token-by-token in the Streamlit interface.
- **Improved retrieval quality tooling** — New easy-to-use reranker presets.
- **Much stronger guidance for large collections** — The Large Vaults guide now includes real benchmarks and clearer scaling advice.

---

## New Features

### Expanded Ingestion Capabilities

PersonalRAGVault now understands many more real-world personal data formats:

| Source              | Format                  | Notes |
|---------------------|-------------------------|-------|
| **WhatsApp**        | `.txt` chat exports     | Individual + group chats, media detection |
| **Notion**          | Markdown exports        | Extracts title + common properties (Created, Tags, etc.) |
| **iCalendar**       | `.ics` files            | Calendar events, meetings, deadlines |
| **Raindrop.io**     | JSON exports            | Bookmarks with folders and tags |
| **Pinboard**        | JSON exports            | Bookmarks with notes and tags |
| **Browser**         | Chrome/Edge HTML        | Full folder hierarchy support |

These additions make it much easier to bring in conversations, saved links, schedules, and notes that live outside traditional note-taking apps.

### Streaming in the Streamlit UI

The web interface now streams LLM responses token-by-token (matching the excellent experience already available in the CLI). This makes chatting with your personal knowledge base feel much more responsive.

### Reranker Presets

Reranking is now easier to use and tune:

- New command: `personalragvault rerankers list`
- Presets via `PRV_RERANK_PRESET`:
  - `tiny` — Fastest, lowest overhead
  - `mini` — Balanced (default)
  - `bge` — Stronger quality

This makes it simpler to get good relevance improvements without having to research model IDs.

---

## Improvements

- **Large Vaults Guide** — Now includes real performance benchmarks (measured on Apple Silicon) for different retrieval modes, chunking strategies, and vault sizes. Includes practical guidance for collections from 5k to 60k+ chunks.
- **Doctor command** — Better dynamic recommendations for larger vaults.
- **Overall UX** — Many small polish items around configuration, diagnostics, and discoverability of advanced features.

---

## Upgrading

```bash
pip install --upgrade personalragvault

# Optional but recommended extras
pip install --upgrade "personalragvault[ui]"
pip install --upgrade "personalragvault[ocr]"
pip install --upgrade "personalragvault[retrieval]"
```

After upgrading, run:

```bash
personalragvault doctor
```

This will give you tailored advice based on your current collection size.
