# PersonalRAGVault v1.2.0

**Major release** — significantly expanded personal data ingestion + better chat experience.

## Highlights

- **New data sources**: WhatsApp chats, Notion exports, iCalendar (.ics), Raindrop.io, and Pinboard bookmarks
- **UI Streaming**: Token-by-token answers now work in the Streamlit web interface
- **Reranker Presets**: Easy `tiny` / `mini` / `bge` presets via `PRV_RERANK_PRESET`
- **Large Vaults Guide**: Now includes real performance benchmarks and scaling advice

## Key Changes

### Added
- WhatsApp chat export parser
- Notion Markdown export parser
- iCalendar (.ics) support
- Raindrop.io + Pinboard bookmark importers
- Streaming support in the Streamlit UI
- `personalragvault rerankers list` command + preset system

### Improved
- Significantly better "Large Vaults" documentation with real benchmarks
- Enhanced `doctor` command recommendations for bigger collections

## Upgrading

```bash
pip install --upgrade personalragvault
# Recommended extras
pip install --upgrade "personalragvault[ui,ocr,retrieval]"
```

After upgrading, run `personalragvault doctor` for tailored advice.
