# Community and discoverability

## GitHub topics

Repository maintainers can add topics in **Settings → General → Topics** or:

```bash
gh repo edit seanebones-lang/personal-RAG \
  --add-topic rag,chromadb,ollama,local-first,privacy,streamlit,personal-knowledge,retrieval-augmented-generation,python,cli
```

Suggested topics: `rag`, `chromadb`, `ollama`, `local-first`, `privacy`, `streamlit`, `personal-knowledge`, `retrieval-augmented-generation`.
